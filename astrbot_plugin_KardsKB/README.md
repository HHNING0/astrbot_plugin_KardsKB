# astrbot_plugin_KardsKB

AstrBot Kards 卡牌游戏开包插件。

## 功能

| 命令 | 说明 |
|------|------|
| `/普通卡包` | 开启现役卡包（5张），保底一张铜卡 |
| `/预备卡包` | 开启预备卡包（5张），保底一张铜卡 |
| `/军官卡包` | 开启军官卡包（7张），保底一张铜卡 + 一张银/金卡 |
| `/卡牌数` | 查看各稀有度卡牌数量统计 |
| `/kards reload` | 重新同步在线卡牌数据并下载到本地 |

## 数据来源

插件优先从 [KARDS Decker](https://1939.giaory.xyz/) 获取在线卡牌数据，成功获取后自动下载卡牌图片到本地 `cards/` 目录缓存的 JPEG 文件。若无法连接则回退使用本地文件

## 卡牌目录

```
cards/
├── Tie/     # 铁卡 (Standard)
├── Tong/    # 铜卡 (Limited)
├── Yin/     # 银卡 (Special)
└── Jin/     # 金卡 (Elite)
```

卡牌图片以 `{card_id}.jpg` 命名，由插件自动同步管理
